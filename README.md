# 《樂樂物語》The Merry Fields

A cozy farming RPG inspired by Story of Seasons — grow crops, raise animals, and build a life in a charming world.

## Features

- 🌱 Seasonal farming — plant, tend, and harvest across four seasons
- 🐄 Animal raising — befriend and care for a variety of lovable animals
- 🏘️ Village life — build relationships with quirky townsfolk
- 🎮 Full controller support — also playable with keyboard & mouse
- 💻 Cross-platform — available on Windows & macOS
