## 介面說明

```
┌──────────────────────────────────────────┐
│ [時間面板]              [小地圖]          │
│  06:00                                   │
│  春 第1天                                │
│ G 500                                    │
│                                          │
│                                          │
│ ████░░░░  ← 體力條                       │
│         [鋤][壺][種]  ← 工具列           │
```

| UI 元素 | 位置 | 說明 |
|---------|------|------|
| 時間面板 | 左上 | 當前時刻與日期 |
| 金幣 (G) | 左上，時間下方 | 持有金幣數量 |
| 體力條 | 左下 | 綠色=充足，紅色=低於 30% |
| 工具列 | 下方中央 | 目前選中工具以亮框標示 |
| 小地圖 | 右上 | 農場全覽，白點為玩家位置 |

## 農耕系統

### 基本流程

```
泥土 → [鋤頭] 翻土 → [澆水壺] 澆水 → [種子] 播種 → 每天澆水 → [Z] 採收
```

### 工具說明

| 工具 | 圖示 | 對應動作 |
|------|------|----------|
| 鋤頭 | 鋤 | 對泥土使用：翻土 |
| 澆水壺 | 壺 | 對翻土/種植格使用：澆水 |
| 種子 | 種 | 對翻土/澆水格使用：播種（蕪菁） |

### 土地狀態

| 外觀 | 狀態 | 可做的事 |
|------|------|----------|
| 棕色淺 | 泥土 | 鋤頭翻土 |
| 棕色中 | 翻土 | 澆水 / 播種 |
| 棕色深 | 澆水 | 播種（或等隔天自然乾） |
| 棕色深 + 植株 | 已種植 | 每天澆水 |
| 棕色中 + 植株 | 已種植（乾） | 需要澆水！ |
| 植株有白點閃光 | 成熟 | 使用 Z 採收 |

> 聚焦在可操作的格子時，格子會顯示**淡綠色半透明框**提示。

### 作物生長

| 作物 | 生長天數 | 售價 | 備註 |
|------|----------|------|------|
| 蕪菁 | 5 天 | 60 G | - |
| 馬鈴薯 | 7 天 | 80 G | - |
| 番茄 | 10 天 | 100 G | 成熟後可多次採收 |
| 南瓜 | 12 天 | 120 G | - |

- 連續 **3 天**未澆水 → 作物枯死
- 成熟後未採收不會枯死，可繼續等待
- 澆水後土地隔天會恢復乾燥外觀，代表今天尚未澆水

### 出貨與收入

1. 採收後物品自動放入**出貨箱**（農場右側）
2. 隔天早上系統自動結算，金幣加入帳戶
3. HUD 左上角 **G 數字**即時更新

## 地圖移動

| 移動方向 | 方法 |
|----------|------|
| 農場 → 村莊 | 走到農場**右側**大門出口 |
| 村莊 → 農場 | 走到村莊**左側**出口 |

場景切換時會有淡入淡出效果。

---

## 時間系統

- 遊戲時間流速：現實 **1 秒 ≈ 遊戲 1 分鐘**
- 一天從 **06:00** 開始
- 日期顯示格式：`季節 第N天`（例：春 第3天）
- 一季 **28 天**，四季循環（春 / 夏 / 秋 / 冬）

---

## 體力系統

- 最大體力：**100**
- 使用工具消耗體力，體力耗盡無法繼續操作
- 體力條顏色：綠色（充足）/ 紅色（低於 30%）
