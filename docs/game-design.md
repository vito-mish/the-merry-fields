# 《樂樂物語》The Merry Fields — Game Design Document

> 一款輕鬆可愛的農場 RPG，支援搖桿與鍵鼠，跨平台 Windows / macOS。

---

## 1. 時間系統

- 一天分三個時段：**早（06:00）、晚（18:00）、夜（22:00）**
- 遊戲時間流速：現實 1 秒 ≈ 遊戲 1 分鐘（可調整）
- **01:00** 開始體力懲罰（隔天最大體力 -20%）
- **04:00** 強制暈倒送醫，扣除醫療費
- 一季 **28 天**，四季循環（春 / 夏 / 秋 / 冬）
- 季節影響：可種植作物種類、天氣機率、NPC 活動

---

## 2. 農耕系統

### 基本流程
```
翻土 → 播種 → 施肥（可選）→ 澆水 → 等待生長 → 收成
```

### 規則
- 未澆水連續 **3 天** 作物枯死
- 雨天自動澆水（無需手動）
- 施肥可加速生長或提升品質（普通 / 優良 / 精品）
- 部分作物可多次收成（不需重新種植）

### 作物品質
| 等級 | 條件 | 售價倍率 |
|------|------|----------|
| 普通 | 基本照顧 | ×1.0 |
| 優良 | 施肥或好天氣 | ×1.5 |
| 精品 | 施肥 + 技能加成 | ×2.0 |

---

## 3. 動物系統

### 動物種類
雞、牛、羊、兔、豬

### 每日照顧
- 餵食（飼料或牧草）
- 梳理（刷毛）
- 放牧（戶外時間提升產品品質）

### 成長系統
- 動物有**年齡**，幼年 → 成年 → 老年
- 成年後可產出商品（蛋、牛奶、羊毛等）
- 老年動物產量下降，可選擇販賣

---

## 4. 體力系統

- 初始最大體力：**100**
- 01:00 後睡覺：隔天體力 -20%（最低 60%）
- 04:00 強制暈倒：送醫扣費，隔天體力 -40%

### 消耗體力的動作
| 動作 | 體力消耗 |
|------|----------|
| 翻土 | -3 |
| 澆水 | -2 |
| 施肥 | -1 |
| 收成 | -1 |
| 梳理動物 | -2 |
| 揮砍（礦洞） | -4 |
| 釣魚（拉竿） | -3 |

### 體力恢復
- 睡覺：完全恢復
- 食物：部分恢復

---

## 5. 社交系統

- 村民 NPC：**20 人**
- 每週同一 NPC 只能送禮 **1 次**
- 戀人關係達成後，每天可送禮

### 好感度等級
| 等級 | 愛心數 | 解鎖內容 |
|------|--------|----------|
| 陌生 | 0–2 | 基本對話 |
| 朋友 | 3–5 | 特殊對話、任務 |
| 好友 | 6–7 | 家中可拜訪 |
| 戀人 | 8–9 | 每日送禮、約會 |
| 結婚 | 10 | 同居、協助農場 |

---

## 6. 經濟系統

### 收入來源
- 販賣農產品、動物製品
- 完成村民委託
- 礦洞挖掘礦石

### 支出
- 商店購買種子、飼料、裝飾
- 工具升級
- 建築擴建（動物欄、儲藏室）
- 送醫費用

### 工具升級
| 工具 | 等級 | 效果 |
|------|------|------|
| 鋤頭 | 1–4 | 翻土範圍 1×1 → 3×3 |
| 澆水壺 | 1–4 | 澆水範圍 1×1 → 十字形 |
| 斧頭 | 1–4 | 礦洞攻擊力提升 |
| 釣竿 | 1–4 | 釣魚範圍與稀有度提升 |

---

## 7. 探索系統

### 礦洞
- 位於森林深處
- 多層結構（每層敵人更強）
- 掉落礦石、寶箱、稀有素材
- 簡單即時戰鬥（揮砍 / 翻滾閃避）

### 釣魚
- 釣點：河邊、海邊
- 小遊戲：計時拉竿
- 魚種依季節、時段、天氣不同
- 稀有魚可賣高價或贈送 NPC

---

## 8. 地圖規劃

### 世界地圖（5 個區域）

```
[ 森林 ] ── [ 玩家農場 ] ── [ 村莊 ]
               │                │
           [ 河岸 ]         [ 礦洞（多層）]
```

| 區域 | 大小（格） | 主要內容 |
|------|------------|----------|
| 玩家農場 | 60×60 | 耕地、動物欄、房屋、儲藏室 |
| 村莊 | 80×50 | 20 個 NPC、商店、廣場、醫院 |
| 森林 | 70×70 | 採集、礦洞入口、隱藏事件 |
| 礦洞 | 30×30 × N層 | 戰鬥、礦石、寶箱 |
| 河岸／海邊 | 80×30 | 釣魚、景觀、季節活動 |

---

## 9. 技術規格

- 引擎：**Godot 4**
- 語言：**GDScript**
- 視角：**2D 俯視（Top-down）**
- 解析度：**320×180**（縮放至 1280×720）
- 平台：Windows / macOS / Steam
- 操作：搖桿 / 鍵盤滑鼠
